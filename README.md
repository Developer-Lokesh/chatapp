Chat App (React.js, Node.js, Express.js, SQL (MYSQL)

About
    This is a simple real-time chat application where users can communicate with each other.
    The project is built using React, Node.js, Express, and SQL Database.

Features
    User Signup & Login
    Real-time Messaging
    One-to-One Chat
    Online/Offline Status
    Responsive Design

Tech Stack
  Frontend:
      React.js
      Tailwind CSS

  Backend:
      Node.js
      Express.js

Database:
    SQL (MySQL)

Other Tools:
    Socket.io (for real-time communication)
    JWT (for authentication)
    Bcrypt (for hashing)
